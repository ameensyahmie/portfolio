Power soalan tu, bro ! Kalau kau nak masukkan font sendiri (custom font) dalam website portfolio kau, ada 2 cara utama ikut mana satu style kau nak:

Option 1: Guna Google Fonts (Senang & Free)

Contoh: Guna Poppins dari Google Fonts

  1. Pergi ke: https://fonts.google.com/

  3. Klik font tekan "Select style" ambil link

Contoh kod:

html
<!-- Tambah dalam <head> -->
<link href="https://fonts.googleapis.com/css2?family=Poppins&display=swap" rel="stylesheet">

<!-- Dalam CSS -->
body {
    font-family: 'Poppins', sans-serif;
}

Kelebihan: Mudah & cepat, tak perlu upload file Kekurangan: Perlukan internet (load dari Google)

Option 2: Upload Font Sendiri (Contoh .ttf atau .woff)

Langkah:

  1. Upload font file ke folder yang sama (contoh fonts/Poppins-Regular.ttf)

  2. Tambah dalam CSS:

css
@font-face {
    font-family: 'PoppinsCustom';
    src: url('fonts/Poppins-Regular.ttf') format('truetype');
}

body {
    font-family: 'PoppinsCustom', sans-serif;
}

Kelebihan: Offline & betul-betul custom
Kekurangan: Kena jaga saiz file dan compatibility
